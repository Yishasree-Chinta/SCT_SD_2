# 🎮 Number Guessing Game

A simple **Number Guessing Game** built using **Python** and **Tkinter** as part of my **Software Development Internship at SkillCraft Technology**.

## ✨ Features

- ✅ Random number generation
- ✅ User-friendly Tkinter GUI
- ✅ "Too High" and "Too Low" hints
- ✅ Attempt counter
- ✅ Play Again option
- ✅ Input validation and error handling

## 🛠️ Technologies Used

- Python
- Tkinter
- Random Module

## 🚀 How to Run

1. Clone the repository.
2. Run `number_guessing_game.py`.
3. Start guessing the number!
